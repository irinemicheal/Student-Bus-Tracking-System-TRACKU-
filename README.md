# 🚀 TRACK U - Student Bus Tracking System

TRACK U is a full-stack web application built using PHP and MySQL that enables efficient student transportation management with role-based access for Admin, Driver, Student, and Parent.

---

## 💡 Features

- 🔐 Role-based login system (Admin, Driver, Student, Parent)
- 📍 Bus tracking system (simulated)
- 🗺️ Route and bus management
- 👨‍👩‍👧 Parent monitoring system
- 📊 Admin dashboard
- 💾 MySQL database integration (CRUD operations)

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: XAMPP (Apache)

---

## ▶️ How to Run

1. Install XAMPP  
2. Place project in `htdocs`  
3. Start Apache & MySQL  
4. Open: http://localhost/studentbustracking/bustracking/

---

## 🎯 Project Purpose

This project demonstrates full-stack development, database integration, and real-world problem solving in transportation tracking systems.
